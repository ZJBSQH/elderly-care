方法一：使用 IDEA Database 面板（推荐）

✅ 推荐理由：
- 图形化界面，操作直观
- 集成在 IDEA 中，无需额外工具
- 支持数据编辑、SQL 执行、表结构查看
- 适合日常开发使用

1.1 打开 Database 面板

1 在 IDEA 右侧边栏找到 "Database" 标签
2 如果没有看到，点击菜单栏 View → Tool Windows → Database

1.2 创建数据库连接

1 点击 Database 面板左上角的 "+" 号
2 选择 "Data Source" → "MySQL"
3 填写连接信息：
配置项 值 说明
Name elderly_mysql 自定义连接名称
Host localhost 数据库主机地址
Port 13306 端口（注意：不是默认的3306）
User root 用户名
Password Zheng666 密码
Database 留空或填写具体库名 可选，留空显示所有数据库
4 点击 "Test Connection" 测试连接
⚠️ 注意：
如果提示缺少驱动，点击 "Download missing driver files" 自动下载 MySQL 驱动
5 测试成功后点击 "OK" 保存连接

1.3 查看数据库和表

1 在 Database 面板中展开连接节点
2 展开数据库列表，可以看到所有数据库： 
elderly_mysql
├── db_elderly_admin      # 后台管理模块数据库
├── db_elderly_auth       # 认证授权模块数据库
├── db_elderly_health     # 健康管理模块数据库
├── db_elderly_medicine   # 用药管理模块数据库
├── db_elderly_news       # 新闻资讯模块数据库
├── db_elderly_remind     # 提醒服务模块数据库
├── db_elderly_user      # 用户管理模块数据库
├── nacos_config         # Nacos配置中心数据库
├── mysql                # MySQL系统数据库
├── performance_schema   # 性能监控数据库
└── sys                  # 系统工具数据库
3 展开某个数据库，可以看到表、视图、存储过程等
4 双击表名，可以查看表数据

1.4 执行 SQL 查询

1 右键点击数据库连接或某个数据库
2 选择 "New" → "Query Console"
3 在打开的 SQL 编辑器中输入查询语句： 
-- 查看所有数据库
SHOW DATABASES;

-- 切换到某个数据库
USE db_elderly_user;

-- 查看所有表
SHOW TABLES;

-- 查看表结构
DESC user;

-- 查询表数据
SELECT * FROM user LIMIT 10;

-- 查看表创建语句
SHOW CREATE TABLE user;
4 点击编辑器上方的绿色运行按钮 ▶ 执行 SQL

1.5 查看表结构

1 在 Database 面板中，右键点击表名
2 选择 "Jump to DDL" 或按 Ctrl+B
3 可以查看表的完整 DDL 语句

1.6 查看表数据

1 双击表名，自动打开数据查看器
2 或右键点击表名，选择 "View Data"
3 可以进行筛选、排序、分页等操作

方法二：使用命令行（Docker 方式）

💡 适用场景：
- 快速查询，无需打开图形工具
- 自动化脚本
- 远程服务器操作

2.1 进入 MySQL 容器

# 查看运行中的容器
docker ps

# 进入 MySQL 容器
docker exec -it elderly-mysql bash

# 或者直接进入 MySQL 命令行
docker exec -it elderly-mysql mysql -uroot -pZheng666

2.2 常用查询命令

-- 查看所有数据库
SHOW DATABASES;

-- 切换数据库
USE db_elderly_user;

-- 查看当前数据库的所有表
SHOW TABLES;

-- 查看表结构
DESC user;
-- 或
DESCRIBE user;

-- 查看表创建语句
SHOW CREATE TABLE user;

-- 查询表数据（前10条）
SELECT * FROM user LIMIT 10;

-- 查看表的字段信息
SHOW COLUMNS FROM user;

-- 查看表的索引
SHOW INDEX FROM user;

-- 查看表的状态信息
SHOW TABLE STATUS LIKE 'user';

-- 查看数据库大小
SELECT 
    table_schema AS '数据库',
    ROUND(SUM(data_length + index_length) / 1024 / 1024, 2) AS '大小(MB)'
FROM information_schema.tables
GROUP BY table_schema;

2.3 退出 MySQL

-- 退出 MySQL 命令行
exit;

-- 或按快捷键
Ctrl + D

4.2 查看表文件

# 进入某个数据库目录
cd db_elderly_user

# 查看表文件
ls -la

# 输出示例：
# user.frm      # 表结构文件（MySQL 5.7及以下）
# user.ibd      # 表数据文件（InnoDB）
# user.MYD      # 表数据文件（MyISAM）
# user.MYI      # 索引文件（MyISAM）
说明：
- .frm 文件：存储表结构定义（MySQL 5.7及以下版本）
- .ibd 文件：InnoDB 表的数据和索引文件
- .MYD 文件：MyISAM 表的数据文件
- .MYI 文件：MyISAM 表的索引文件
- MySQL 8.0+ 使用 InnoDB 存储引擎，每个表对应一个 .ibd 文件

常用查询语句速查表

数据库相关

-- 查看所有数据库
SHOW DATABASES;

-- 查看当前使用的数据库
SELECT DATABASE();

-- 切换数据库
USE db_elderly_user;

-- 创建数据库
CREATE DATABASE db_test DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 删除数据库
DROP DATABASE IF EXISTS db_test;

-- 查看数据库创建语句
SHOW CREATE DATABASE db_elderly_user;

表相关

-- 查看所有表
SHOW TABLES;

-- 查看表结构
DESC user;
DESCRIBE user;

-- 查看表创建语句
SHOW CREATE TABLE user;

-- 查看表的字段信息
SHOW COLUMNS FROM user;

-- 查看表的索引
SHOW INDEX FROM user;

-- 查看表的状态
SHOW TABLE STATUS LIKE 'user';

-- 查看所有表的大小
SELECT 
    table_name AS '表名',
    ROUND(((data_length + index_length) / 1024 / 1024), 2) AS '大小(MB)'
FROM information_schema.TABLES
WHERE table_schema = 'db_elderly_user'
ORDER BY (data_length + index_length) DESC;

数据查询

-- 查询所有数据
SELECT * FROM user;

-- 查询前10条
SELECT * FROM user LIMIT 10;

-- 条件查询
SELECT * FROM user WHERE status = 1;

-- 排序查询
SELECT * FROM user ORDER BY create_time DESC;

-- 分页查询（第1页，每页10条）
SELECT * FROM user LIMIT 0, 10;

-- 统计查询
SELECT COUNT(*) FROM user;

-- 分组统计

用户和权限

-- 查看当前用户

-- 授权
GRANT ALL PRIVILEGES ON db_elderly_user.* TO 'test'@'%';

-- 刷新权限

常见问题解决

问题1：连接被拒绝

错误信息： Access denied for user 'root'@'localhost'

解决方法：
1. 检查用户名和密码是否正确
2. 检查端口是否正确（13306）
3. 检查 MySQL 容器是否启动：docker ps
4. 检查防火墙是否开放端口

问题2：找不到 Database 面板

解决方法：
1. 确认使用的是 IntelliJ IDEA Ultimate 版本（社区版可能没有此功能）
2. 点击菜单 View → Tool Windows → Database
3. 或安装插件：Database Navigator

问题3：无法连接到数据库

解决方法：
1. 确认 Docker 容器已启动：docker ps | grep mysql
2. 确认端口映射正确：docker port elderly-mysql
3. 尝试重启容器：docker restart elderly-mysql
4. 查看容器日志：docker logs elderly-mysql

问题4：中文乱码

解决方法：
1. 在连接 URL 中添加字符集参数：
jdbc:mysql://localhost:13306/db_name?useUnicode=true&characterEncoding=utf8mb4
2. 检查数据库字符集：SHOW VARIABLES LIKE 'character%';
3. 检查表字符集：SHOW CREATE TABLE table_name;
